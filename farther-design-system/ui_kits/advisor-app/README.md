# Advisor workspace UI kit

A high-fidelity mock of the in-product advisor experience. Built from brand
guidelines — colors, type, spacing, components all traced to the tokens in
`colors_and_type.css`.

Includes:
- Dark rail navigation with charcoal ground and bronze notification pip
- Sticky search bar, command palette affordance
- KPI row (4-up)
- Stacked bar chart (book growth by source)
- Top-clients table with tag pills + tabular numerics
- Task list, donut allocation, IW opportunity card (dark inverse)

If there's an actual Farther advisor codebase, point me at it and I'll retune components to match.
